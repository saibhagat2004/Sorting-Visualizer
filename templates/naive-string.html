Naive-String algorithm

The naive string matching algorithm, also known as the Brute Force Algorithm, is a straightforward approach for searching a pattern or substring within a larger text.

In Naive-String Matching algorithm,

  • The algorithm starts by sliding the pattern over the text one character at a time. It compares each character of the text with the corresponding characters of the pattern.
  • At each position, it checks if the characters match. If a match is found (i.e., all characters of the pattern match the corresponding characters in the text), it reports the occurrence of the pattern at that index.
  • If no match is found, the algorithm slides the pattern by one position and repeats the process. It continues this sliding and checking until the entire text has been examined.
  • Whenever a complete match is found, the algorithm prints the index where the pattern occurs in the text.

  • Its best-case time complexity is O(n), when the pattern is found at the very beginning of the text.
